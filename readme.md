# cheri-berry

---

A lightweight KNative NPM private repository

# How to develop

```bash
# Install dependencies
go mod download

# cd to the cmd directory
cd cmd

# Run the server
go run main.go
```

# Features

- [x] npm registry
- [x] npm publish
- [x] npm install
- [x] npm adduser
- [x] npm whoami
- [x] npm unpublish
